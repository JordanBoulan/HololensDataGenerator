Class PAVDataGenerator

source code

SET ALL CONSTANTS HERE BEFORE RUNNING OTHERWISE DEFAULT VALUES WILL BE USED Previously generated values are used to create a more realistic next value. Adjust how many packets are created per generation. When the system sends all packets in the set it starts fresh and generates a new set of packets
